源码编译

Python 官网可以下载到源码。

Linux 上编译需要先安装额外模块，例如 Ubuntu

Windows 上只需打开 PC/VS9.0/pcbuild.sln，选择最少的模块 python, pythoncore 编译，如果这两个模块编译时报错，只需按照错误提示，钩上所需模块即可。

Mac 上只需要

本系列文章在 Mac 上完成，但环境本身只影响编译，并不影响分析过程，读者选择方便阅读源码的环境即可。
